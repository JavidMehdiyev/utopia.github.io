Website Preparation

“How Website Works?” - was our third homework, where our goal was to create a website using HTML,CSS, and JavaScript. Firstly, we decided to give a certain name to our team. After long discussion we stopped at “Utopia”.

Secondly, we decided to make beautiful design to our website and logotype following minimalism. So we decided to take a black, white and lavender color in navigation menu. Design of website was created with Figma graphic editor, which is used for web design.

After designing the website’s each page, we started to write the code on HTML, CSS, and Javasript, adding animations on buttons, and making it more creative and beatiful. Website consist of several details: Main page with our team name, navigation bar, Hour of Code project, Website preparation and Team members. In each of these pages we describe the certain topic as Hour of code project details, personal information about each of the team members, and how was the website prepared. In general this homework was very interesting for us, and gave more experince in coding and design.
